## 정의
데이터 처리 속도가 엄청 빠른 데이터를 Key-Value 형태로 저장하는 NoSQL 데이터베이스이다.
## 장점
레디스는 인메모리(RAM)에 모든 데이터를 저장하기 때문에 데이터의 처리 성능이 굉장히 빠르다.

> RDBMS의 데이터베이스는 대부분 디스크(Disk)에 데이터를 저장한다.
> 디스크보다 메모리에서의 데이터 처리 속도가 월등하게 빠르다.
## 사례
- 캐싱
- 세션 관리
- 실시간 분석 및 통계
- 메시지 큐
- 자리공간 인덱싱
- 속도 제한
- 실시간 채팅 및 메시징

현업에서는 **캐싱**(데이터 조회 성능 향상)이 많이 사용된다.
