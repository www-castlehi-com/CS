### vs [[데코레이터 패턴]]
#### 공통점
- 프록시 사용
#### 차이점
- 프록시 패턴 : **접근 제어**
- 데코레이터 패턴 : **새로운 기능 추가**
## 기능
### 1️⃣ 접근 제어
- 권한에 따른 접근 차단
- 캐싱
- 지연 로딩
### 2️⃣ 부가 기능 추가
- 원래 서버가 제공하는 기능 + 부가 기능
